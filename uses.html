Distribution

My current Linux distribution is Void Linux because I like its idea, minimalism and package manager.

Plus, it doesn't use nearly as many resources as Debian did, which literally gave me free heating.

Void Linux's official website: voidlinux.org

Editor

I've been a devoted Vim user for a while before finally discovering Neovim. I've decided to switch mainly because of the speed upgrade and overall UX.

My config is all pure Vimscript, no plugins and very small, even featuring a neat statusbar! I'll upload it whenever I upload my full dotfiles.

Neovim's official website: neovim.io

Environment

Despite me preferring newer solutions over older ones, I still use X11! I've also taken a liking to half-obscure window managers. For example, for the last month I've been hopping from evilWM to CTWM, which are both surprisingly well-performing in 2024. EvilWM is still maintained, the last release being March this year (2024).

My terminal is usually just urxvt. Not much to be said here, simple and sweet. That's it.
