The lemon bar

I've been a devoted X11 user for as long as I can remember, and I usually have a bar sitting at the top of my screen.

For the 2 months that I've used spectrwm, that bar was provided by the window manager itself, but now that I'm on cwm, I need to find a bar myself.

And that bar is none other than Lemonbar by LemonBoy!

Lemonbar is a simple x11 bar that just draws a window somewhere on the screen and exits. Simple enough.

Now here's the interesting part: it doesn't have any sort of config. At all. No modules, no presets, nothing. Instead, you feed it text from stdin. That can be done by echoing the text into the bar itself manually or through a shell script.

I've thrown together an example which you can see here:

#!/usr/bin/env bash

while true; do
        echo "Hi! The date is $(date +'%I:%M %p')"
        sleep 0.2
done

Customization

For now, I recommend getting the xft fonts fork of lemonbar from here.

All customization is done through flags given at bar startup. The full list can be found on the manpage, but here's a couple:

-g: changes bar geometry
-B: changes background color
-F: changes foreground color
-u,-U: changes underline width and color, respectively
-F: changes default font

Those alone should be enough to get you up and going. For more stuff you can do with it, check out Sacules' guide on it here.

Happy tweaking!
